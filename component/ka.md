# 卡證

## 掃碼 / QR Code {#qr-code}

目前台北卡有六種卡別：普通卡、愛心、愛陪、學生、敬老、優待（未來會陸續增加）。掃碼出現在個人頁當中，讓使用者可以透過掃碼取代實體卡片的身份認證。
